# Blog Era Grab-Bag — Design Spec

**Date:** 2026-04-27
**Status:** Approved

## Overview

The blog (`scene_blog`) currently renders a single static WordPress Kubrick (2009) era. This spec upgrades it into a grab-bag of 8 web design eras: the first visit always shows the 2024 AI slop era, and every subsequent navigation reshuffles randomly into a different era from the pool.

## Structure

### New file: `html/js/scenes-blog.js`

`scene_blog()` moves out of `scenes-entry.js` into a new dedicated file. All 8 era render functions live there. A new `<script src="js/scenes-blog.js"></script>` tag is added to `index.html` immediately before `router.js` (load order: after all other scene files, before router).

Remove the existing `scene_blog()` definition from `scenes-entry.js` and the `.wp-post`, `.wp-meta`, `.wp-title`, `.wp-excerpt` CSS classes from `index.html` (they move into inline styles in the new file).

### State

- `S.blogVisits` (number, initialized to `0` in the `S` object in `core.js`) — incremented at the top of `scene_blog()` on every call.
- `S.lastBlogEra` (string, initialized to `''`) — stores the key of the last rendered era to prevent immediate repeats.

### Era selection logic

```
scene_blog():
  incDepth()
  S.blogVisits++
  if S.blogVisits === 1:
    render era: 'aislop'
    S.lastBlogEra = 'aislop'
  else:
    pick random era from pool excluding S.lastBlogEra
    render chosen era
    S.lastBlogEra = chosen era key
```

## Navigation changes

All links inside the blog that previously pointed to `signin` now point to `blog`, so every click that looks like navigation reshuffles the era:

- "Read more »" → `blog`
- Author name links → `blog`
- Category/tag links → `blog`
- "Newer posts / Older posts" pagination → `blog`
- Tag cloud links → `blog`
- Sidebar "Recent Posts" links → `blog`

Links that stay unchanged:
- "Return to Nexus" / home link → `main`
- Sidebar search button → toast (existing behavior)
- Sidebar meta links (Log in → `signin`, others → `footer_dead`)

`incDepth()` fires on every `scene_blog()` call (unchanged from current behavior).

## The 8 Eras

All eras render the same 4 blog posts conceptually (future of productivity, 10 synergy tips, legacy tools, hello world), adapted in tone and chrome to match the period.

### 1. 2024 — AI Slop *(first visit)*

- Purple/indigo gradient branding; "NexusAI Content Engine™" badge in header
- Post titles bloated to ~15 words ("Unlocking Synergistic Productivity: A Comprehensive Deep-Dive Into the Future of Collaborative Work Ecosystems")
- Each post metadata: "Generated by NexusAI · 2,400 words · 12 min read · Fact-checked by our team*"
- Footer asterisk: "*Team review pending. This content was created to help synergize your workflow engagement journey."
- Clean white/light grey background, Inter/system-ui font
- "Subscribe for AI-curated insights" CTA in sidebar

### 2. 2015 — Medium Style

- Full-width light grey hero banner area
- Giant serif (Georgia) headline, very large
- Floating left-side social share bar: ♥ · 💬 · bookmark icons
- "X min read" pill next to author name
- Circular author avatar placeholder
- Clean white background, generous line height, no sidebar

### 3. 2013 — Bootstrap Flat

- Dark `#343a40` navbar with brand name and nav links
- Flat `#007bff` "Read More" buttons
- 8-column main / 4-column sidebar Bootstrap-style grid
- Breadcrumb trail below navbar
- Helvetica/Arial, clean grey palette (#f8f9fa background)
- Sidebar has "Categories" widget with badge counts

### 4. 2009 — WordPress Kubrick

- Existing implementation moved verbatim to the new file
- No content changes

### 5. 2006 — Web 2.0

- Glossy gradient header (blue, `linear-gradient`)
- Orange BETA badge next to blog title
- Orange RSS chicklet button
- Each post has: "Digg this! | del.icio.us | StumbleUpon" share row
- ★★★★☆ star rating display on each post
- Rounded corners (`border-radius`) on all widgets
- "AJAX-powered" search label in sidebar

### 6. 2003 — LiveJournal / Early Blogger

- Pastel background (#e8e8d0)
- Purple muted header bar
- Post metadata includes: `current mood: productive 📊 | music: lo-fi beats to synergize to`
- Earnest, lowercase, slightly confessional tone in excerpts
- Long blogroll sidebar: list of ~8 fake "Friends" sites as links → `footer_dead`
- "Leave a comment" links and "Friends Only 🔒" padlock label

### 7. 1998 — GeoCities

- Dark background with repeating diagonal stripe or starfield pattern (CSS `repeating-linear-gradient`)
- `font-family: 'Comic Sans MS', cursive` throughout
- Header: `✨ THE NEXUS BLOG ✨` in yellow
- `🚧 UNDER CONSTRUCTION 🚧` banner below header
- Hit counter: `Visitors: 000247`
- "Best viewed in Netscape Navigator 4.0 at 800×600 resolution"
- `<marquee>` tag for a scrolling announcement strip
- Guestbook link → `footer_dead`

### 8. 1995 — Early Web / Mosaic

- `background-color: #c0c0c0` (silver)
- `font-family: Times, 'Times New Roman', serif`
- Blue underlined links, no hover styles
- `<hr>` horizontal rules between posts and sections
- Zero layout — single column, no grid, no sidebar
- Post structure: H2 title, small date line, paragraph text, HR
- Footer: "Last updated: April 4, 2009. This page best viewed with Netscape Navigator 2.0."
- No images; any image placeholder is replaced with `[image]` text

## CSS

The existing `.wp-post`, `.wp-meta`, `.wp-title`, `.wp-excerpt` classes in `index.html` are removed. All styling for all 8 eras is inline within `scenes-blog.js`, consistent with the rest of the codebase pattern.

## `index.html` changes

1. Add `<script src="js/scenes-blog.js"></script>` before `<script src="js/router.js"></script>`
2. Remove the 4 `.wp-*` CSS class definitions from the `<style>` block

## `core.js` changes

Add `blogVisits: 0` and `lastBlogEra: ''` to the `S` state object.

## `scenes-entry.js` changes

Remove the entire `scene_blog()` function (lines 543–600).

## Out of scope

- Individual blog post pages (no `scene_blog_post` scenes)
- Depth-based era selection (eras are random, not tied to `S.depth`)
- Persistent era history across sessions
